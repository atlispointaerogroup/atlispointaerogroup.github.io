Pilot Requirements

Ferry Pilot Application

All pilots operating with AtlisPoint must meet the following minimum requirements before being considered for any assignment.

Airline Transport Pilot License (ATP)

A valid ATP certificate issued by an ICAO-recognized authority is required for all ferry pilot engagements.

Current Type Rating

Pilots must hold a current, valid type rating on the specific aircraft to be ferried. No exceptions.

First Class Medical Certificate

A valid first class medical certificate is required and must remain current throughout the duration of any assignment.
